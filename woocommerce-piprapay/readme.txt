Accept payments using PipraPay gateway in your WooCommerce store.

== Description ==

PipraPay allows merchants in Bangladesh to accept mobile and bank payments on their WooCommerce store using a simple, secure integration.

**Key Features:**

- Accept Bkash, Nagad, Rocket, and bank transfers
- Live and Sandbox environments
- Auto-complete digital product orders
- Webhook support for secure order status updates
- Supports physical and digital products
- Custom checkout logo display
- Seamless integration with WooCommerce

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > Settings > Payments > PipraPay to configure your API keys and settings.

== Frequently Asked Questions ==

= How do I get an API key? =
Visit [https://piprapay.com](https://piprapay.com) and register for a merchant account to get your live/sandbox API keys.

= Does this plugin support test mode? =
Yes, you can enable Sandbox mode in the settings to test your integration before going live.

= What payment methods are supported? =
Bkash, Nagad, Rocket, and traditional bank transfers.

= Does this plugin work for digital and physical products? =
Yes, and digital products can be set to auto-complete after successful payment.
